# simple-Picture-Annotation
单网页图片批注工具\
轻量，高效\
\
【适配】✅firefox类浏览器✅macbook的safari及其触控板❌鼠标滚轮缩放未适配chrome类浏览器\
【基本使用方法】：
- 右键进行拖动，中间视图还原缩放，左键进行标注动作
- shift
  - 切换形状工具等边/等半径
  - 切换线条工具的水平或竖直约束
  - 临时切换画笔工具的直线
- 鼠标中键缩放视图到窗口 （妈妈再也不会担心我找不到画布了）
- 文字工具使用enter进行提交，shift+Enter进行换行，Esc取消

【亮点】
- 画笔工具默认带有平滑，使用鼠标标注的时候，字迹看起来更平滑
- 能够添加文字，而且允许用户换行
- 有基本颜色，用户能够临时增加颜色，并右键删除自定义颜色
- CTRL+V粘贴图片，CTRL+C复制图片（safari因浏览器限制，无法做到）
- 能够撤销
- 点击按钮，一键保存，无需确认
